# ValutaTrade Hub

ValutaTrade Hub — это консольное приложение для симуляции торговли фиатными и криптовалютами. Платформа позволяет пользователям регистрироваться, управлять виртуальными портфелями, совершать сделки по покупке/продаже валют и отслеживать актуальные курсы в реальном времени.
